# 生成 PRD 文档

## Context

用户要求基于当前项目生成 PRD（产品需求文档），作为开发文档使用。

**项目概况**：
- 名称：商业航天情报系统
- 目的：铂力特（688333.SH）每日复盘
- 数据源：Tushare
- 输出：Markdown 日报

**核心功能**：
1. 行情数据线：8步流程，产出 85 个指标
2. 新闻数据线：配置驱动的新闻抓取和事件提取
3. 日报模块：生成结构化复盘报告

## 目标

创建 `docs/prd.md`，包含：
- 产品概述
- 业务需求
- 功能规格
- 数据设计
- 接口规格
- 技术架构
- 开发规划
