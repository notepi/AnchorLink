# 历史分析页面讲解

> 3000/history 的使用指南

---

## 这个页面在做什么

打个比方：气象局不会只看今天温度来预报明天，而是找历史上"气压、湿度、风向和今天类似的日子"，看那些日子之后天气怎么走。

这个页面做的就是这件事——给铂力特(688333.SH)建了一本"病历"。每当铂力特出现某种状态组合（板块涨跌、超额高低、信号出现），就去翻 240 多个交易日的历史记录，找到类似的日子，统计之后怎么走的。

核心逻辑链只有四步：

```
今天什么状态 → 历史上同类状态之后发生了什么 → 概率多大 → 预测准不准
```

页面上的每个区块，都是这条链上的一个环节。下面逐个讲。

---

## 第一课：先弄懂"状态"——9 宫格象限

页面很多地方提到"状态"，指的是 9 宫格象限的位置。

9 宫格由两个维度拼成：

| | 个股强 | 个股中 | 个股弱 |
|---|---|---|---|
| **产业链强** | 强+强 | 强+中 | 强+弱 |
| **产业链中** | 中+强 | 中+中 | 中+弱 |
| **产业链弱** | 弱+强 | 弱+中 | 弱+弱 |

- **产业链强/中/弱**：航天主池（11 只股票）当天中位数收益。涨 >0.5% 为强，跌 >0.5% 为弱，中间为中。
- **个股强/中/弱**：铂力特相对航天主池的超额收益。超额 >+0.5% 为强，<-0.5% 为弱，中间为中。

为什么要拆两个维度？因为"板块涨+个股也涨"和"板块跌+个股逆势涨"是完全不同的市场环境，后续走势也不同。9 宫格就是把这两种信息叠在一起分类。

---

## 第二课：顶部导航——选哪天看

页面顶部是日期选择器。

- **选某一天**：看那天的状态映射、迁移概率、稳定性
- **7 天滑动窗口**：左右箭头切换，快速浏览
- **区间模式**：选一段连续交易日，看区间内的统计汇总

默认显示最新交易日。选历史日期可以回溯验证——"那天系统怎么说，实际怎么走"。

---

## 第三课：历史映射——今天像历史上的哪天

这是页面最核心的区块。

### 为什么要看这个

"映射"就是找相似日。不是找价格相似的，而是找**状态组合**相似的——象限位置一样、信号组合相近的日子。找到之后看那些日子之后的 T+1、T+3、T+5 收益，就是"历史上同类状态之后发生了什么"。

### 怎么读

**当前状态卡**：最上方显示选中日期的象限位置和信号标签。比如"产业链强+个股强"，下面列出当天出现的信号（行业Beta为正、放量上涨等）。

**路径标签**：6 种分类之一，是对相似日后续走势的概括——

| 路径标签 | 含义 |
|---------|------|
| 强势延续 | T+1/3/5 全正且胜率 ≥55% |
| 冲高回落 | T+1 涨但之后回落 |
| 继续走弱 | 多个窗口收益为负 |
| 弱势修复 | 短期有反弹但没守住 |
| 分化震荡 | 涨跌都不大，胜率接近 50% |
| 样本分歧 | 相似日走势不一致 |

**窗口统计表**：这是映射的核心数据——

- **T+1 均值**：相似日次日平均收益
- **T+3 / T+5 均值**：3 天、5 天后的平均收益
- **胜率**：相似日中上涨的比例
- **95% 置信区间**：用 Bootstrap 重采样算的，意思是"如果你遇到 100 次这样的状态，95 次的收益会落在这个范围内"。区间越窄，结论越稳定。

**相似日分布图**：X 轴是历史日期，Y 轴是价格，蓝色点是相似日（越深越大越相似），琥珀色线是铂力特走势。看蓝色点是聚集还是分散——聚集说明历史规律一致，分散说明"历史上类似状态后走势并不统一"。

**最相似案例**：Top 5 相似日的具体日期、匹配度和后续收益。如果 top 5 方向一致，结论可信度更高；如果涨跌都有，说明这条路径不确定。

## 第四课：状态迁移——明天最可能去哪

9 宫格说的是今天的静态位置。但市场是动态的——今天在"强+强"，明天可能还在，也可能滑到"中+强"。

### 9×9 热力图

- **行**：今天在哪个象限
- **列**：明天去了哪个象限
- **颜色深浅**：从今天这个状态迁移到明天那个状态的历史概率

比如行是"强+强"，列是"强+强"的格子最红，说明"强+强 → 强+强"（连续强势）概率最高。

### 路径判定

热力图上方会给出一个文字判定——明天最可能迁移到哪个象限，以及概率。这不是预测涨跌，而是告诉你"状态切换的概率分布"。

### 观察要点

- 对角线（状态不变）通常概率最高，说明状态有惯性
- 如果某行颜色很分散，说明这个状态不稳定，明天去哪都有可能
- 关注偏离对角线的高概率格——那些是"转折信号"

## 第五课：稳定性——最近跟不跟板块走

这个区块回答一个具体问题：铂力特最近是在跟着板块走，还是走出了独立行情？

### 4 个快览卡

| 卡片 | 含义 |
|------|------|
| 跑赢/跑输 | 当天超额是正还是负，具体多少 |
| 关系类型 | 强跟随 / 弱跟随 / 独立 / 反向 |
| 异常度 | 当天偏离是否超出正常范围 |
| 连胜/连输 | 连续几天跑赢或跑输 |

### 滚动超额曲线

5d 超额和 10d 超额是**滚动累加**的：过去 5/10 天每天超额收益之和。

- 曲线在 0 以上 = 最近整体跑赢板块
- 曲线在 0 以下 = 最近整体跑输
- 急速上升/下降 = 短期偏离加速

### 跟随/偏离图

把铂力特收益和板块收益画在一起，直观看到同步还是背离。

### 操作提示

系统会根据当前状态给出简要提示（如"跑输后历史修复概率 62%"）。这是统计描述，不是操作建议。

## 第六课：预测准不准——预测评估

前几课说了"历史上类似状态之后怎么走"，但这个预测到底靠不靠谱？这个区块就是回答这个问题。

### 三个核心指标

| 指标 | 含义 | 好的标准 |
|------|------|---------|
| IC (Spearman 相关系数) | 预测值和实际值的排序相关性 | >0.05 有预测力，>0.1 较强 |
| 方向命中率 | 涨跌方向判断正确的比例 | >55% 优于随机 |
| RMSE | 预测值和实际值的偏差大小 | 越小越好，但要看和收益波动比 |

### 五分位验证

把所有预测日按"预测涨幅"从大到小分 5 组（Q1=最看好，Q5=最不看好），看每组的实际收益。

如果 Q1 实际涨最多、Q5 实际跌最多，说明预测有排序能力。如果 Q1-Q5 实际差不多，说明预测没用。

这是最直观的判断方法。

### 置信区间

T+1 / T+3 / T+5 的 95% 置信区间。比如 T+1 置信区间是 [-1.2%, +2.3%]，意思是历史上 95% 的相似日次日收益落在这个范围。**区间越宽，不确定性越大**。

## 第七课：个性画像——这只股票什么"性格"

如果把铂力特当成一个人，这个区块画的就是它的"性格测试"结果——它在什么环境下表现好、什么环境下容易踩坑。

### 环形图

4 种模式的比例分布：

| 类别 | 含义 |
|------|------|
| 偏好环境 | 出现这类信号时，历史后续表现好 |
| 规避环境 | 出现这类信号时，历史后续表现差 |
| 反直觉机会 | 直觉觉得不好，但实际表现好的信号 |
| 信号陷阱 | 直觉觉得好，但实际表现差的信号 |

**反直觉和陷阱是画像最有价值的部分**——它们揭示的是"市场共识错在哪里"。

### 4 类模式详解

**偏好环境**：比如"行业Beta为正且处于行业前排"这种信号组合出现时，历史 T+1 正收益概率高。出现在这里说明"这些条件对铂力特是利好"。

**规避环境**：反过来，比如"放量大涨"出现时历史后续反而差。可能因为放量冲高后容易回落。

**反直觉机会**：比如"板块跌但个股抗跌"，直觉觉得不好，但历史数据显示这种状态之后反而容易涨。

**信号陷阱**：比如"板块普涨"，直觉觉得好，但铂力特在这种环境反而容易落后。

### 8 个核心指标

| 指标 | 含义 |
|------|------|
| 胜率 | T+1 上涨的比例 |
| 期望值 | 每次操作的数学期望（平均赚/亏多少） |
| 信息比率 | 超额收益的稳定性，越高说明超额越持续 |
| T+3 超额 | 3 天后相对板块的中位数超额 |
| T+3 不利 | 3 天后最差情况（下方风险代理） |
| 盈亏比 | 平均盈利 / 平均亏损的比率 |
| 夏普 | 绝对收益的风险调整后指标 |
| 信号覆盖 | 有信号标注的交易日占比 |

### 板块联动关系

4 个池子各自的联动特征——铂力特是跟随、领先还是独立于每个池子。

- **增材确认池**（3 只）：同赛道最直接的对比
- **航天主池**（11 只）：产业链核心标的
- **航天主题池**（8 只）：情绪和主题驱动标的
- **交易观察池**（9 只）：成交活跃的关联标的

## 第八课：信号时间线——信号随时间怎么变化

这个区块把信号出现的时间和价格走势画在一起，能看到两件事：

### 信号分布

价格走势图上用彩色竖线标记 4 类信号出现的时间：

| 颜色 | 信号类别 |
|------|---------|
| 红色 | 偏好环境 |
| 绿色 | 规避环境 |
| 紫色 | 反直觉机会 |
| 橙色 | 信号陷阱 |

### 信号频率变化

对比"全样本"和"近 20 日"的信号出现频率。如果最近某类信号突然增多，可能意味着市场环境发生了变化。

### 高频信号车道

列出历史上频繁出现的信号，用时间轴显示每次出现的位置。能看出某些信号是阶段性的（比如某个信号只在某几个月密集出现）。

---

## 第九课：折叠区——想深挖再看

页面底部有一个折叠的研究区域，包含更细粒度的数据：

### 象限统计格

9 宫格的完整统计——每个格子的样本数、平均次日收益、胜率。用颜色标识最好和最差的象限。

### 信号提升表

所有信号的排序表——每个信号的历史样本数、次日平均收益、相对基线的提升量（lift）、胜率。按 lift 从高到低排序。

### 信号组合

两个信号同时出现时的叠加效果。有些信号单独看一般，但组合在一起效果显著；有些组合反而抵消。

### 极端背离时间线

历史上个股和板块严重背离的日子——背离幅度、之后 T+1/T+3 的修复情况。用于理解"极端偏离后的回归力度"。

| 交易观察池 | 交易联动与风险映射池 | 9 只 |

### 9 宫格状态

| | 个股强 | 个股中 | 个股弱 |
|---|---|---|---|
| **产业链强** | 强+强 | 强+中 | 强+弱 |
| **产业链中** | 中+强 | 中+中 | 中+弱 |
| **产业链弱** | 弱+强 | 弱+中 | 弱+弱 |

- 产业链强/中/弱 = 航天主池中位数收益 >+0.5% / ±0.5% 之间 / <-0.5%
- 个股强/中/弱 = 铂力特超额 >+0.5% / ±0.5% 之间 / <-0.5%

### 路径标签

| 标签 | 含义 |
|------|------|
| 强势延续 | T+1/3/5 全正且胜率 ≥55% |
| 冲高回落 | T+1 涨但之后回落 |
| 继续走弱 | 多窗口收益为负 |
| 弱势修复 | 短期反弹但没守住 |
| 分化震荡 | 涨跌不大，胜率约 50% |
| 样本分歧 | 相似日走势不一致 |

### 关系类型

| 类型 | 含义 |
|------|------|
| 强跟随 | 个股高度跟随池子波动 |
| 弱跟随 | 有跟随但独立性较强 |
| 独立 | 个股走势与池子基本无关 |
| 反向 | 个股走势与池子相反 |

### 信号类别

| 类别 | 含义 |
|------|------|
| 偏好环境 | 出现时历史正收益显著 |
| 规避环境 | 出现时历史负收益显著 |
| 反直觉机会 | 直觉与实际方向相反 |
| 信号陷阱 | 看似正面实际负面 |

### 关键指标

| 指标 | 全称 | 怎么读 |
|------|------|-------|
| T+1 | Next 1 day return | 次日收益 |
| T+3 | Next 3 day return | 3 天后累计收益 |
| T+5 | Next 5 day return | 5 天后累计收益 |
| IC | Information Coefficient | 预测排序能力，>0.05 有用 |
| 胜率 | Win Rate | 上涨比例 |
| 超额 | Excess Return | 个股收益 - 池子中位数收益 |
| 5d/10d 超额 | Rolling excess | 过去 5/10 天每日超额之和 |
| 置信区间 | Confidence Interval | 95% CI，越窄越确定 |
| RMSE | Root Mean Square Error | 预测偏差，越小越准 |
| Lift | Signal Lift | 有信号时收益 - 无信号时基线收益 |
| 盈亏比 | Payoff Ratio | 平均盈利 / 平均亏损 |

## 重要提醒

1. **样本量有限**：约 240 个交易日，单只股票。某些象限的样本可能只有十几天，结论需谨慎。
2. **前视偏差**：部分阈值使用样本内数据设定，存在前视偏差。
3. **不构成投资建议**：所有分析都是历史统计描述，不预测未来。
4. **收盘后数据**：所有指标基于收盘价计算，盘中不可用。
5. **超额是累加的**：5d 超额不是单日概念，是过去 5 天每日超额之和。一个异常日可以显著影响滚动数值。
