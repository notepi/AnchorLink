# 铂力特股票池重定义方案（稳定版）

> 文件用途：用于指导 `pools.yaml` 的股票池口径重构。  
> 研究对象：688333.SH 铂力特。  
> 生成日期：2026-05-05。  
> 适用系统：AnchorLink / TradingTS / 铂力特每日复盘模块。  
> 核心原则：不新增股票池，只重定义现有四个池子的职责与成员。

---

## 0. 一句话结论

铂力特股票池不应该做成“增材制造 + 商业航天 + 军工电子”的百科清单，而应该重定义为：

> **以商业航天硬科技链为主 benchmark，以臻镭科技为第一高 beta 风向标，以华曙高科 / 超卓航科作为增材制造本业校验，以中国卫通 / 上海瀚讯等作为商业航天主题温度计，以鸿远电子 / 振华科技 / 火炬电子作为军工资金风格观察。**

最终目标不是回答“哪些公司和铂力特有关系”，而是回答：

> **铂力特今天为什么涨跌？是商业航天硬科技主线共振，还是主题情绪，还是军工电子风险偏好，还是个股独立事件？**

## 1. 对当前 `pools.yaml` 的重新理解

当前文件采用三层结构：

```text
anchor       -> 锚定标的：688333.SH 铂力特
instruments  -> 证券主数据
universes    -> 股票池定义
memberships  -> 股票与股票池的多对多关系
```

这个结构是合理的，不需要推翻。

问题不在文件结构，而在**股票池口径**：

1. `industry_chain` 当前偏泛产业链，没有明确成为商业航天硬科技主池；
2. `direct_peers` 过薄，只有华曙高科，无法稳定支撑本业 benchmark；
3. `theme_pool` 基本合理，但应明确为主题温度计，不参与 benchmark；
4. `trading_watchlist` 捕捉到了军工电子联动，但缺少近期最关键的 688270 臻镭科技；
5. 航天动力、航天电子等标的的 `role` 与真实业务口径不够准确。

## 2. 铂力特当前股价的核心解释框架

铂力特现在至少有四条解释线：

| 解释线 | 作用 | 对股价影响 |
|---|---|---|
| 商业航天硬科技主线 | 火箭、卫星、星载电子、结构件、发动机材料的产业链共振 | 最重要，决定估值弹性 |
| 增材制造本业 | 金属 3D 打印设备、材料、定制化制造服务的行业强弱 | 用来验证铂力特是否有本业独立逻辑 |
| 工业 / 3C 批产 | OPPO、荣耀、汽车、热管理、机器人等民用工业扩散 | 决定业绩兑现能力，但不单独建池 |
| 军工电子 / 高 beta 交易联动 | 臻镭科技、鸿远电子、振华科技、火炬电子等短期资金风格 | 决定短线波动解释力 |

其中，股票池主轴应按下面顺序排列：

```text
第一主轴：商业航天硬科技链
第二主轴：臻镭科技 / 军工电子高 beta 交易联动
第三主轴：增材制造本业校验
第四主轴：商业航天主题情绪扩散
```

## 3. 四个股票池重新定义

### 3.1 `direct_peers`：增材制造本业确认池

#### 新定义

`direct_peers` 不再被理解为“主行情解释池”，而是用于回答：

> 铂力特今天的涨跌，是否来自 3D 打印 / 增材制造本业共振？

#### 使用原则

- 只放真正与铂力特业务相近的增材制造标的；
- 不放商业航天、卫星通信、军工电子标的；
- 可参与 benchmark，但权重不能高于 `industry_chain`；
- 样本小于 2 时，不输出“本业均值”，只输出“单一可比观察”。

#### 建议成员

| symbol | name | 操作 | role | benchmark | 理由 |
|---|---|---|---|---|---|
| 688433.SH | 华曙高科 | 保留 | `additive_equipment_peer` | 是 | 工业级 3D 打印设备直接可比，是铂力特本业校验的核心标的 |
| 688237.SH | 超卓航科 | 新增 | `additive_service_peer` | 是 | 航空增材制造 / 维修再制造属性，与高端制造链更接近 |
| 300227.SZ | 光韵达 | 可选新增 | `weak_additive_proxy` | 否 | 有 3D 打印和激光加工属性，但纯度和财务质量弱于前两者 |

#### 推荐参数

```yaml
universe_id: direct_peers
display_name: 增材制造本业确认池
purpose: "判断铂力特是否存在增材制造本业共振，而非商业航天或军工情绪带动"
can_be_benchmark: true
min_size: 2
benchmark_type: additive_manufacturing_core
```

### 3.2 `industry_chain`：商业航天硬科技主池

#### 新定义

`industry_chain` 应该成为铂力特股票池的主池。它不再是泛泛的上下游池，而是：

> 商业航天硬科技链主 benchmark，用于判断火箭、卫星、星载电子、结构件、发动机材料、连接器等硬件链是否向铂力特传导。

#### 使用原则

- 这是主 benchmark；
- 只放商业航天硬科技链，不放纯应用层主题票；
- 允许不是铂力特直接上下游，但必须能解释商业航天硬件链景气；
- 有 ST / 监管风险的标的可以进入，但必须限权重并打风险标签。

### 3.4 `trading_watchlist`：交易联动与风险映射池

#### 新定义

`trading_watchlist` 不是产业池，而是短线解释池。它回答：

> 铂力特今天是不是被臻镭科技、军工电子、高 beta 商业航天票、监管风险映射带动？

#### 使用原则

- 不参与 benchmark；
- 可与其他池子复用同一只股票；
- 重点看近 5 日、近 10 日、近 20 日相关性；
- 必须区分 `business_relevance` 和 `trading_correlation`，避免把交易相关误写成业务相关。

#### 建议成员

| symbol | name | 操作 | role | 理由 |
|---|---|---|---|---|
| 688270.SH | ST臻镭 | 新增，第一优先级 | `high_beta_space_risk_signal` | 商业航天电子链高 beta，同时具备 ST / 信披风险映射 |
| 603267.SH | 鸿远电子 | 保留 | `military_electronics_signal` | 军工电子高相关交易信号 |
| 000733.SZ | 振华科技 | 保留 | `military_electronics_signal` | 军工电子高相关交易信号 |
| 603678.SH | 火炬电子 | 保留 | `military_electronics_signal` | 军工电子高相关交易信号 |
| 688102.SH | 斯瑞新材 | 保留 | `space_material_trading_signal` | 航天材料高 beta，高相关交易信号 |
| 300762.SZ | 上海瀚讯 | 可复用 | `satcom_high_beta_signal` | 商业航天通信高 beta |
| 301005.SZ | 超捷股份 | 可复用 | `rocket_structure_high_beta_signal` | 商业火箭结构件高 beta |

#### 推荐字段

当前文件只有：

```yaml
relevance: 0.90
```

建议拆成：

```yaml
business_relevance: 0.20
trading_correlation: 0.90
corr_window: 20
corr_asof: "2026-05-05"
```

这样可以避免日报中出现错误表述，例如：

> 错误：鸿远电子与铂力特产业链高度相关。  
> 正确：鸿远电子与铂力特短线交易联动强，主要反映军工电子风险偏好。

## 6. 日报解释规则

后续日报不要只输出“铂力特 vs 板块均值”，而应该输出四类判断。

| 判断项 | 使用股票池 | 解释 |
|---|---|---|
| 本业确认 | `direct_peers` | 铂力特是否跟 3D 打印 / 增材制造同业共振 |
| 主线确认 | `industry_chain` | 商业航天硬科技链是否整体强化 |
| 情绪扩散 | `theme_pool` | 商业航天 / 卫星互联网主题是否普涨 |
| 交易联动 | `trading_watchlist` | 是否被臻镭科技、军工电子、高 beta 风格带动 |

### 6.1 状态标签规则

| 市场表现 | 状态标签 | 解释 |
|---|---|---|
| 铂力特涨，ST臻镭涨，`industry_chain` 强 | 商业航天硬科技主线共振 | 最强信号，说明市场在交易商业航天硬件链 |
| 铂力特涨，`theme_pool` 强，但 `industry_chain` 一般 | 主题情绪带动 | 可能只是商业航天题材扩散，基本面确认不足 |
| 铂力特涨，鸿远 / 振华 / 火炬强 | 军工电子风险偏好带动 | 短线交易属性强，持续性需观察 |
| 铂力特跌，ST臻镭因风险大跌 | 监管风险映射 | 不是产业主线走弱，而是信披 / ST 风险映射 |
| 铂力特涨，华曙 / 超卓也强 | 增材制造本业确认 | 说明 3D 打印本业可能被市场重估 |
| 铂力特强，所有池子弱 | 个股独立事件驱动 | 需要优先查公告、机构调研、订单、资金异动 |
| 铂力特弱，`industry_chain` 强 | 个股风险压制 | 重点检查监管、估值、财报质量、减持、机构卖出 |

## 8. 最终口径

重构后的股票池应该服务一个核心问题：

> **铂力特近期股价到底是在交易什么？**

因此，不建议把股票池做成行业百科；也不建议新增 3C、材料、低空经济、机器人等新池。工业 / 3C 是铂力特业绩兑现的重要解释项，但当前股票池的主线仍应围绕商业航天。

最终稳定口径：

```text
铂力特 = 商业航天硬科技主线
        + 增材制造本业验证
        + 臻镭科技高 beta 联动
        + 军工电子资金风格
        + 监管风险映射
```

四个股票池的职责为：

```text
direct_peers       -> 本业确认
industry_chain     -> 主线确认
theme_pool         -> 主题温度
trading_watchlist  -> 交易联动 / 风险映射
```

这套定义比原始版本更适合用于铂力特每日复盘、相对强弱分析、交易状态解释和后续自动报告生成。
