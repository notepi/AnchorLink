# 一、分析框架总览

## 1.1 分析目标

预测铂力特 T+1 超额收益（vs 产业链中位数），而非绝对收益。绝对收益被行业 Beta 主导，超额收益才代表个股真实 Alpha。

## 1.2 三层分析架构

```
第一层：行业链比较（AnchorLink 核心，不可替代）
├── A. 状态路径（9象限转移）
├── B. 信号初现效应
├── G. 相对强度分位（rs_vs_chain）
├── H. 量价背离（anchor × moneyflow × amount）
├── J. 连续跑赢/跑输 streak
├── K. Alpha 信号排行（超额目标重排）
├── M. 超额均值回归（excess_5d/10d）
└── N. 池联动相关

第二层：市场状态识别（Regime 过滤，决定信号效力）
├── S. ADX Regime（mean_reverting / trending / transition）
└── CUSUM Regime（辅助方向确认）

第三层：价格模式确认（验证方向，不提供独立 Alpha）
├── R. 经典技术指标（RSI / MACD / BB / Stoch / ADX）
├── V. 波动率体系（ATR / BB带宽 / Squeeze）
├── U. 季节性效应（周内 / 月份）
└── T. SMC 信号（FVG / BOS / LiqSweep）
```

**核心逻辑**：第一层提供 Alpha 来源，第二层决定何时用/不用信号，第三层确认方向。单独第三层几乎无用，但三层交叉后效力翻倍。

# 二、铂力特的核心特性

## 2.1 均值回归股

**所有"看起来强"的信号都是减仓信号。** 这是铂力特最根本的特性，被 21 个维度无例外地验证：

| 维度 | "看起来强"的信号 | 实际 T+1 超额 | 胜率 |
|------|----------------|-------------|------|
| H 维 | 放量大涨 | -0.79% | 38% |
| R 维 | MACD 金叉 | +0.28%（无效） | — |
| R 维 | Stoch 金叉 | -0.78% | 42.5% |
| R 维 | BB 上轨触及 | -1.26% | 36.8% |
| R 维 | RSI 超买 | -1.67% | 35.3% |
| T 维 | BOS 创 20 日新高 | -1.42% | 39.3% |
| S 维 | transition+streak≥+2 | -0.97% | 25.0% |

**操作含义**：铂力特的买点是"看起来弱"——缩量阴跌、MACD 死叉、streak 跑输、RSI 偏低。追涨必亏。

## 2.2 超额半衰期极短

excess_5d 自相关半衰期仅 2 天（A 股长期均值 232 天）。5 天超额进入极低档后 T+1 平均反弹 +1.94%。这意味着超额收益的均值回归非常快，操作窗口短。

## 2.3 Regime 决定一切

同一信号在不同 Regime 下效力天差地别：

| 信号 | mean_reverting | trending | transition |
|------|---------------|----------|------------|
| streak≤-3 | **+3.11% / 77.8%** | +0.68% / 50% | +1.60% / 71.4% |
| streak≤-2 | **+1.91% / 65.0%** | +0.03% / 45.5% | +0.75% / 66.7% |
| streak≥+2 | -0.47% / 43.8% | -0.01% / 48% | **-0.97% / 25.0%** |

streak≤-2 全局看无效（46%），但 MR 环境下是强信号（65%）。V1 把它当废信号，是被 trending 的噪音稀释了。

# 四、综合评分规则（V2 保守，生产推荐）

## 4.1 评分计算

每日评分 = 所有触发信号的权重之和（买入为正，卖出为负）。

### 买入信号权重

| 信号 | 权重 | 触发条件 |
|------|------|---------|
| MR + streak≤-3 | +5 | ADX≤20 且 outperform_streak≤-3 |
| MR + streak≤-2 | +3 | ADX≤20 且 outperform_streak≤-2 |
| TS + streak≤-2 | +3 | 20<ADX<25 且 streak≤-2 |
| excess_5d P15- | +3 | excess_5d ≤ 训练集 P15 |
| excess_10d P15- | +3 | excess_10d ≤ 训练集 P15 |
| streak≤-3（非 MR） | +3 | streak≤-3 且 ADX>20 |
| 跌但资金支撑 | +2 | anchor<0 & moneyflow>0.6 |
| delta_excess_5d 温和回调 | +2 | -3.1 ≤ Δexcess_5d ≤ -1.0 |
| alpha 信号≥3 | +2 | 纯 Alpha 信号数≥3 |
| path 弱→强 | +2 | 昨弱+今中/强 |
| MACD 柱负 | +1 | MACD histogram < 0 |
| 周三效应 | +1 | 星期三 |
| LiqSweep 高 | +1 | 假突破上方阻力 |

# 五、操作指南

## 5.1 买入操作

```
1. 计算 Regime（ADX 三态）
2. 计算综合评分
3. 判断：
   - MR 环境：score ≥ 3 → 做多
   - TR/TS 环境：score ≥ 4 → 做多
4. 仓位：
   - MR + score≥5：70%
   - MR + score 3-4：50%
   - TR + score≥4：40%
   - TS + score≥4：30%
```

## 5.2 卖出/回避操作

```
1. Veto 条件（直接不做多）：
   - 放量大涨（anchor>1% & amount>1.3）

2. 强回避：
   - 周五 + ADX<25 → 绝对不做多
   - TS + streak≥+2 → 减仓

3. 弱回避：
   - 周五 → 降低仓位
   - RSI>70 / Stoch>80 / BB%b>1 → 保守
   - 看跌FVG → 降低仓位
```

## 5.3 持仓周期

| 信号 | 持仓 | 理由 |
|------|------|------|
| MR + streak≤-3 | 3-5 天 | T+3 100%胜率，T+5 87.5% |
| MR + streak≤-2 | 2-3 天 | T+3 75%胜率 |
| 其他买入信号 | 1 天 | 无持续证据 |
| 所有卖出信号 | 仅 T+1 | 卖出信号衰减极快 |

## 5.4 不该做的事

| 禁忌 | 原因 |
|------|------|
| 追涨（MACD 金叉、Stoch 金叉、创新高买入） | 均值回归股追涨必亏 |
| 周五做多 | 36.7%胜率，周末避险效应 |
| 看跌FVG 后继续持有 | 信号 T+2 就失效 |
| TS 环境下低阈值做多 | 50%胜率，噪音太多 |
| 相信 kNN 相似度预测 | 48.5%命中率，不如随机 |

# 六、分析方法论

## 6.1 分析流程

```
数据准备
├── 4池行情 → anchor_return, industry_beta, anchor_alpha
├── 16维特征计算 → A-Q
├── 5维新增特征 → R/V/U/S/T
└── Regime 标记 → ADX三态

信号验证
├── 单信号 lift + 胜率（基线）
├── Regime × 信号 交叉（核心）
├── Bootstrap CI（鲁棒性）
└── 信号衰减（T+1→T+5）

策略构建
├── 加权评分
├── Regime 自适应阈值
└── Walk-Forward 回测（120天滚动窗口）

验证
├── Monte Carlo（子集重采样）
├── 评分极端日验证
└── 信号冗余分析（phi 系数）
```

## 6.2 关键方法论原则

1. **目标变量用超额收益**，不用绝对收益。绝对收益被 Beta 污染。
2. **Regime 先于信号**。不区分 Regime 的信号分析会被噪音稀释。
3. **Walk-Forward 验证**。120 天滚动窗口，所有阈值从训练集计算，消除前视偏差。
4. **更多信号 ≠ 更好策略**。V3 比 V2 多了 CUSUM 和 FVG veto，效果反而更差。信号贵精不贵多。
5. **小样本结论不可信**。n<20 的信号只能标记方向，不能用于评分。
6. **Bootstrap 不显著不代表无效**。21 个信号 0 个通过严格检验，但方向 100% 一致（"强"=卖）。这本身就是信息。

## 6.3 统计诚实度

| 问题 | 事实 | 影响 |
|------|------|------|
| OOS 仅 127 天 | 统计结论需更长时间验证 | 可能过拟合 |
| 无严格统计显著性 | Bootstrap CI 大多跨零 | 不能说"确认有效" |
| 夏普 6.58 | 31 个交易日算出来的 | 实际会更低 |
| Monte Carlo P(wr>50%)=97.4% | 还有 2.6% 可能是噪音 | 不是 100% |
| 日历效应仅 1 年 | 8 月/4 月效应需 2-3 年验证 | 可能是年度特例 |

**底线**：所有结论基于 247 天数据，方向一致但统计力度不足。随数据积累，部分信号可能被证伪。
