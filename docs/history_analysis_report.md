## 十、总结

1. **铂力特是反转型标的**：弱势状态反而是较好的介入时机，强势状态要警惕回落
2. **"行业强但个股弱"是最危险象限**：次日 -0.50%，胜率 42.1%，且更可能恶化而非修复
3. **信号体系是均值回归型**："坏信号"反而是买入参考，"好信号"反而要警惕
4. **极端背离全部是个股事件**：背离后 1-3 天有修正但难回均衡
5. **up_ratio 相关性不稳定**：当前 0.4-0.5 是阶段性抬升，不能当稳定因子
6. **样本量有限**：96 天、1 只标的，统计结论需谨慎，应持续积累数据验证
