# AnchorLink 术语表

> 所有文档和代码中涉及池名称、状态标签的展示文本，必须以本表为准。
> 修改展示术语前先改本文件，再同步到 `web/src/lib/glossary.ts`。

---

## 池名称

| 池 ID (代码用) | 短名称 (UI 展示) | 全称 (首次出现/tooltip) | 能否做 benchmark |
|---------------|----------------|----------------------|----------------|
| `direct_peers` | 增材确认池 | 增材制造本业确认池 | 是 |
| `industry_chain` | 航天主池 | 商业航天硬科技产业链主池 | 是 |
| `theme_pool` | 航天主题池 | 商业航天主题情绪池 | 否 |
| `trading_watchlist` | 交易观察池 | 交易联动与风险映射池 | 否 |

### 口径规则

1. **"主线池"** 仅用于 benchmark 计算时的合并基准（direct_peers + industry_chain），不作为任何单一池的独立名称
2. 九宫格中 industry_beta 的展示文本用"产业链强/中/弱"，不用"行业强/中/弱"
3. 任何新文案提到池子时用短名称；首次出现或需要解释时用全称

## 状态标签

### industry_beta

| 代码值 | 展示文本 | 含义 |
|--------|---------|------|
| `positive` | 产业链强 | 航天主池中位数收益 > +0.5% |
| `neutral` | 产业链中 | 航天主池中位数收益在 ±0.5% 之间 |
| `negative` | 产业链弱 | 航天主池中位数收益 < -0.5% |

### anchor_alpha

| 代码值 | 展示文本 | 含义 |
|--------|---------|------|
| `positive` | 个股强 | 个股相对产业链超额 > +0.5% |
| `neutral` | 个股中 | 个股相对产业链超额在 ±0.5% 之间 |
| `negative` | 个股弱 | 个股相对产业链超额 < -0.5% |

### risk_level

| 代码值 | 展示文本 | 含义 |
|--------|---------|------|
| `low` | 低风险 | 数据完整，无异常信号 |
| `medium` | 中风险 | 数据部分缺失或池内分化严重 |
| `high` | 高风险 | 数据不足或存在联动背离 |

## 九宫格状态组合

| 代码键 | 展示文本 |
|--------|---------|
| positive+positive | 产业链强+个股强 |
| positive+neutral | 产业链强+个股中 |
| positive+negative | 产业链强+个股弱 |
| neutral+positive | 产业链中+个股强 |
| neutral+neutral | 产业链中+个股中 |
| neutral+negative | 产业链中+个股弱 |
| negative+positive | 产业链弱+个股强 |
| negative+neutral | 产业链弱+个股中 |
| negative+negative | 产业链弱+个股弱 |

## 路径标签

| 代码值 | 展示文本 | 条件 |
|--------|---------|------|
| `strong_rise` | 强势延续 | T+1/3/5 收益全正 + 胜率全 ≥55% |
| `pullback_after_rise` | 冲高回落 | T+1 涨 >0.5% 但 T+1 与 T+5 差 ≥0.7% |
| `continue_fall` | 继续走弱 | 2+ 窗口收益 < -0.3% |
| `weak_repair` | 弱势修复 | T+1 或 T+3 >0.3% 但 T+5 不正或胜率 <50% |
| `range_bound` | 分化震荡 | 所有窗口收益绝对值 <0.2% 且胜率 45-55% |
| `disagreement` | 样本分歧 | 以上都不满足 |

---

## 联动关系

| 代码值 | 展示文本 | 含义 |
|--------|---------|------|
| `strong_follow` | 强跟随 | 个股高度跟随池子波动 |
| `weak_follow` | 弱跟随 | 有一定跟随但独立性较强 |
| `independent` | 独立 | 个股走势与池子基本无关 |
| `inverse` | 反向 | 个股走势与池子相反 |
