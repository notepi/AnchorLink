## 七、non_overlapping 是否支持主结论？

5D Q5-G4 T+5 对比：

| 模式 | n | 超额中位数 | 超额胜率 |
|------|---|-----------|---------|
| all_signals | 11 | -2.14% | 45% |
| non_overlapping | 7 | +2.19% | 57% |

**non_overlapping 方向反转。** 这说明 all_signals 中的负超额可能被重叠样本放大。Q5-G4 的 T+5 结论不稳定，应降级为"无信号"。

其他主要格子（热端+上升、冷端+下降）的 non_overlapping 方向与 all_signals 一致，主结论成立。

### 关键回答

1. **5D Q×G 比单独 Q 更有解释力**：热端细分后，"热端+下降"（-2.41%）比"热端+上升"（-1.88%）负超额更大，说明超额方向（G）在热端有增量信息
2. **10D Q×G 比 5D 更有区分力**：10D 热端+上升的负超额 -2.48% 且 Anchor 自身微跌，信号更清晰
3. **Q5 极热 + G上升 = 危险追高**：不是 Anchor 下跌，而是指数涨更多（5D）或 Anchor 走弱（10D）
4. **Q5 极热 + G下降 = 已降温但非均值回归**：5D 热端+下降超额中位 -2.41% 比"热端+上升"的 -1.88% 更差，说明极热位置超额下降不是降温而是持续回落
5. **Q1 极冷 + G下降 = 接飞刀**：短期负超额，T+5 可能均值回归但不确定
6. **Q1 极冷 + G回升 = 弱均值回归启动**：5D Q1-G4 的 T+5/T+10 正超额来自 Anchor 自身反弹
7. **usable 与 strict_ok_only 基本一致**
8. **non_overlapping 支持主结论**，但 Q5-G4 的 T+5 不稳定，应降级为无信号
